# Visão Geral do Projeto

## 🎯 Objetivo
Bot assistente para gerenciamento de agenda que integra com Google Calendar usando linguagem natural em português.

## 🚀 Funcionalidades
- Consultar agenda
- Verificar disponibilidade
- Criar eventos
- Gerenciar compromissos

## 🛠️ Tecnologias
- Python 3.9+
- Flask
- Google Calendar API
- OAuth2
- HTML/CSS/JS
